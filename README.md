# Task 2 - Sentiment Analysis using Machine Learning and Deep Learning

## Project Overview

This project performs Sentiment Analysis on movie reviews using both Machine Learning and Deep Learning techniques. The objective is to classify reviews as **Positive** or **Negative** and compare the performance of different models.

---

## Objective

The main objective of this project is to build and evaluate sentiment analysis models using:

- Logistic Regression
- Support Vector Machine (SVM)
- Long Short-Term Memory (LSTM)

---

## Dataset

**Dataset Used:** IMDb Movie Reviews Dataset

The dataset contains movie reviews labeled as:

- Positive
- Negative

---

## Technologies Used

- Python
- TensorFlow / Keras
- Scikit-learn
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Google Colab

---

## Machine Learning Models

### Logistic Regression
- TF-IDF Vectorization
- Binary Classification
- Performance Evaluation

### Support Vector Machine (SVM)
- LinearSVC Classifier
- TF-IDF Features
- Performance Comparison

---

## Deep Learning Model

### LSTM (Long Short-Term Memory)

The LSTM model consists of:

- Embedding Layer
- LSTM Layer
- Dropout Layer
- Dense Output Layer
- Sigmoid Activation

---

## Evaluation Metrics

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- Classification Report
